# 🌍 小世界探索家 - 幼儿园地图学习器

一款专为3-6岁幼儿设计的互动地理学习应用。以温暖友好的"小探险家"为主题，通过可爱卡通化的世界地图，引导孩子们踏上环球探险之旅。

## ✨ 功能特点

### 🗺️ 探索模式
- 可缩放、拖拽的交互式世界地图
- 点击国家显示详细信息卡片（国旗、名称、首都）
- 大洲切换按钮，高亮显示对应区域
- 语音播放功能（中文发音）

### 🎮 游戏模式
三种适合小朋友的游戏类型：

1. **国旗识别游戏** - 显示国旗，选择对应的国家名称
2. **首都匹配游戏** - 显示首都名称，选择对应的国家国旗
3. **大洲分类游戏** - 显示国旗，选择所属大洲

### 📚 学习模式
- 按大洲分类浏览所有国家
- 每个国家显示国旗、名称和首都
- 点击可查看详细信息

### ⭐ 个人中心
- 统计收集的星星数量
- 学习过的国家数量
- 游戏次数和正确率
- 成就徽章系统

## 🎨 设计特点

- **童话绘本风格**：柔和渐变背景、圆润边角、可爱图标
- **调色板**：薄荷绿主色调、阳光黄、珊瑚红等温暖色彩
- **字体**：ZCOOL KuaiLe（童趣标题）+ Noto Sans SC（清晰正文）
- **动效**：弹性动画、点击反馈、成功/错误提示动画
- **响应式设计**：适配平板、桌面和移动设备

## 🚀 使用方法

1. 克隆仓库
```bash
git clone https://github.com/buptcxy/kindergarten-map-learner.git
cd kindergarten-map-learner
```

2. 启动本地服务器
```bash
python3 -m http.server 8888
```

3. 打开浏览器访问 `http://localhost:8888`
