# 幼儿园地图学习器 - 设计规范

## 1. 概念与愿景

**小世界探索家** - 一款专为3-6岁幼儿设计的互动地理学习应用。以温暖友好的"小探险家"为主题，通过可爱卡通化的世界地图，引导孩子们踏上环球探险之旅。界面如同一个巨大的游乐场，每个国家都是一个小站点，等待着小朋友们去探索和发现。整体氛围轻松愉快，让学习变成一场有趣的冒险。

## 2. 设计语言

### 美学方向
采用"童话绘本"风格 - 柔和的渐变背景、圆润的边角、手绘质感的图标元素。整体如同翻阅一本精心绘制的儿童绘本。

### 调色板
- **主色调**: #4ECDC4 (薄荷绿 - 代表探索与自然)
- **次要色**: #FFE66D (阳光黄 - 代表快乐与活力)
- **强调色**: #FF6B6B (珊瑚红 - 用于重要交互)
- **辅助色**: #95E1D3 (浅绿)、#F38181 (粉红)、#AA96DA (淡紫)
- **背景色**: #FFF9E6 (奶油白) 渐变至 #E8F6F3 (薄荷白)
- **文字色**: #5D5D5D (深灰) / #FFFFFF (白色)

### 字体
- **标题**: "ZCOOL KuaiLe" (站酷快乐体) - 童趣可爱
- **正文**: "Noto Sans SC" - 清晰圆润
- **字号**: 最小24px，标题32-48px

### 空间系统
- 基础间距单位: 16px
- 按钮最小尺寸: 64x64px (触摸友好)
- 卡片圆角: 24px
- 大触控区域: 不小于 88x88px

### 动效哲学
- **入场动画**: 弹性放大 (spring effect), 400ms
- **点击反馈**: 轻微缩放 + 水波纹效果
- **成功反馈**: 星星绽放 + 彩带飘落
- **切换动画**: 平滑滑动, 300ms ease-out
- **悬停效果**: 柔和发光 + 轻微上浮

### 视觉资产
- 使用 Inline SVG 绘制所有图标和地图元素
- 国旗使用简化卡通版本
- 地图元素采用圆润、可爱的绘制风格

- 平板优先 (768px - 1024px)

## 4. 功能与交互

### 4.1 探索模式
- **世界地图浏览**: 可缩放、拖拽的交互式世界地图
- **国家点击**: 点击国家显示卡片 (国旗+名称+首都)
- **大洲切换**: 点击大洲按钮高亮该区域
- **语音播放**: 点击播放当前选中国家的名称发音

### 4.2 游戏模式
- **国旗匹配**: 显示国旗，从3个选项中选择对应国家
- **首都配对**: 显示国家名称，从3个选项中选择正确首都
- **地理问答**: "找出在亚洲的国家"等简单问答
- **奖励机制**: 每答对1题获得1-3颗星星

### 4.3 学习模式
- **分类学习**: 按大洲浏览国家
- **详细卡片**: 每个国家的国旗、名称、首都、位置
- **跟读功能**: 播放标准中文发音，可跟读

### 4.4 交互细节
- **点击反馈**: 元素缩放至0.95，150ms后恢复
- **成功反馈**: 绿色勾选动画 + "太棒了！" 语音
- **错误反馈**: 轻微摇晃 + "再试一次" 提示
- **加载状态**: 可爱的加载动画 (旋转的地球)

## 5. 组件清单

### 5.1 导航栏 (NavBar)
- 高度80px，奶油色背景
- Logo: "🌍 小世界探索家" 文字Logo
- 模式切换: 胶囊形toggle按钮
- 进度显示: 星星图标 + 数量
- 状态: 默认/游戏模式高亮

### 5.2 底部导航 (BottomNav)
- 高度100px，包含安全区
- 4个导航项，每项88x88px触摸区域
- 图标+文字布局
- 状态: 未选中(灰色) / 选中(主题色+放大图标)

### 5.3 国家卡片 (CountryCard)
- 尺寸: 280x360px
- 内容: 国旗(160x100px) + 国名(大字) + 首都 + 语音按钮
- 圆角24px，柔和阴影
- 状态: 默认 / 悬停(上浮+阴影加深) / 展开(放大至全屏)

### 5.4 地图区域 (MapView)
- SVG交互式世界地图
- 国家区域可点击
- 缩放级别: 1x - 3x
- 手势: 点击、拖拽、捏合缩放

### 5.5 游戏卡片 (GameCard)
- 尺寸: 200x240px
- 包含国旗或地理元素图片
- 状态: 默认 / 选中(边框高亮) / 正确(绿色勾) / 错误(红色叉)

### 5.6 进度条 (ProgressBar)
- 星星收集进度可视化
- 每次获得星星时有飘落动画
- 显示: "12/50 ⭐"

### 5.7 语音按钮 (SpeakButton)
- 尺寸: 64x64px
- 波浪动画指示正在播放
- 点击后播放标准中文发音

## 6. 技术方案

### 框架选择
- **纯HTML5 + CSS3 + Vanilla JavaScript**
- 理由: 最快加载速度，无构建依赖，适合儿童应用

### 核心实现
- SVG地图: 使用简化版世界地图SVG
- 语音合成: Web Speech API (中文发音)
- 动画: CSS3 Animations + Transitions
- 存储: LocalStorage (保存进度)

### 数据结构
```javascript
const country = {
  id: "cn",
  name: "中国",
  nameEn: "China",
  capital: "北京",
  flag: "🇨🇳", // 使用Emoji简化
  continent: "asia",
  position: { x: 75, y: 35 } // 地图上的大致位置
}
```

### 性能优化
- 地图使用懒加载
- 图片使用SVG内联
- CSS动画使用transform/opacity
- 首次加载 < 2秒

### 无障碍设计
- 所有交互元素可键盘访问
- ARIA标签支持
- 高对比度模式支持
- 触摸反馈明显
